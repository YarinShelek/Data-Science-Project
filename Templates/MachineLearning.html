In this project, we tried to predict a player's rank using given stats on him.
The question is obviously a regression-type question, so we went ahead and tried regression-based algorithms for the prediction.
The algorithms we tried, were: K Nearest Neighbors, Support Vector Classifier, Decision Tree, Neural Network, Random Forest, Naive Bayes
As you can see, our model works the best with Neural Network, so we chose it for the predictions.
